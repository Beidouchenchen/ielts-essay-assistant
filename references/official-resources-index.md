## 如何使用这些资源

### 步骤1: 获取官方范文
从上述资源中下载/复制你需要的官方范文

### 步骤2: 添加到个人语料库
使用 skill 的个人语料库功能：
```
你: 我要添加一篇官方范文到语料库
→ 提供范文内容
→ 系统自动分析并存储
```

### 步骤3: 分析学习
Skill 会分析官方范文的：
- 结构模式 (Structure)
- 高分词汇 (Lexical Resource)
- 句式变化 (Grammatical Range)
- 衔接技巧 (Cohesion)

### 步骤4: 应用生成
基于学习内容生成新话题的范文
